About the website

This website was built using the R package blogdown, which incorporates R Markddown into the Hugo framework. The design of the website is based on the template hugo universal, originally developed by Bootstrapious and ported to Hugo by DevCow. The animated globe is a modification of a script by Joan Perals (https://codepen.io/jperals/pen/bWjPPg).

Some of the code was directly written in HTML, CSS and Javascript (e.g. the CV and about page, ToC). All code (.Rmd) is available in the github repository saskiaotto/mds_blog.
